EcoHunters

Built with a theme of sustainability at its heart, EcoHunters is meant for users to pinpoint areas of environmental distress in their relative locations. For example, different items such as "Hazard" or "Waste" can be pinpointed to a user's location - these points from multiple users will accumulate and can be further analyzed using the heatmap feature, which will allow local organizations and communities to identify areas that need to be addressed.

  • Created - April 2023
  • technologies used - Python, Django, Bootstrap, Google Maps API, HTML, CSS, Javascript
  • Role - Full Stack Developer
  • View Online - https://devpost.com/software/ecohunter
